# ANTI-SLOP.md — AI 味检测清单（小说创作专用）

> 本文件服务于 autonovel-workflow，用于在初稿审阅阶段系统性地检测和清除「AI 味」文本。
> 与自媒体文案的去 AI 味检测不同，本清单聚焦于**小说叙事**特有的虚假感、模式化和情感空洞。

---

## 一、词汇级检测（正则可扫描）

以下词汇和句式模式可通过 grep / ripgrep 批量扫描标记。

### 1.1 AI 偏爱词汇（黄色警告）

| 词汇/短语 | 典型上下文 | 建议替代 |
|-----------|-----------|---------|
| 在当下这个... | "在当下这个时代/社会/世界" | 直接切入具体情境 |
| 不禁让人思考/不禁让人感叹 | "看着这一切，不禁让人思考……" | 删除，或用角色行动体现 |
| 以一种……的方式 | "以一种近乎残忍的方式" | 直接描写，去掉"以……的方式"框架 |
| 某种/某种程度上的 | "某种莫名的恐惧" "某种程度上的释然" | 要么具体，要么删除 |
| 似乎/仿佛/好像/或许 | 每个角色都"似乎"如何如何 | 只在有必要表示不确定性时使用 |
| 突然/忽然 | 每页出现多次 | 保留必要处，其余用具体动作替代 |
| 就在这时 | "就在这时，门开了" | 直接写"门开了" |
| 一般/一样/似的 | "如释重负一般" "死一般的寂静" | 删掉"一般"，直接写"死寂" |
| 某种说不清道不明的 | "某种说不清道不明的感觉" | 直接定义那是什么感觉 |
| 就这样 | "就这样，他离开了" | 去掉"就这样" |

### 1.2 禁区词（红色警告 — 应根除）

| 禁区词 | 原因 |
|--------|------|
| 在这个充满变数的时代 | AI 式陈词滥调 |
| 不禁让人深思/反思 | 说教式过渡 |
| 命运似乎/仿佛在开玩笑 | 滥用的命运隐喻 |
| 也许这就是人生吧 | 空洞哲理 |
| 时间仿佛在这一刻静止了 | AI 最爱的时间定格描述 |
| 空气中弥漫着一种……的气氛 | 懒人写法，直接写感官细节 |

### 1.3 句式单一检测

- **主谓一致检测**：每个段落都以「他/她/它/这座城/这个房间」开头 → 变化句式
- **副词过多**：每个动词前加副词（突然地、迅速地、缓缓地、轻轻地） → 选一个只保留 20%
- **「是…的」结构泛滥**："他是愤怒的""她是疲惫的" → 换成主动描写

## 二、句式级检测

### 2.1 Show-Don't-Tell 违规

**检测方法**：搜索情感词汇的直接陈述。

| ❌ Tell（标记） | ✅ Show（推荐） |
|---------------|---------------|
| 他很愤怒。 | 他握紧了拳头，指节发白。 |
| 她很伤心。 | 眼泪在眼眶里转了转，她没让它掉下来。 |
| 他感到恐惧。 | 他的后背贴紧了墙壁，呼吸变得又浅又急。 |
| 这个地方看起来很破旧。 | 墙皮卷成干枯的叶子，露出下面的灰砖。 |

**高频 Tell 词**：感到、觉得、看起来、似乎显得、充满了、弥漫着、发现自己是……的

### 2.2 陈述句过度

**症状**：连续 3 句以上以「X 是 Y」/「X 有 Y」结构开头。

> ❌ 他是这个城市最年轻的警探。他有十年的经验。他是局里最不守规矩的人。
> 
> ✅ 这个城市最年轻的警探，十年经验，局里出了名的不守规矩。

**检测**：扫描"是/有/存在/有着/属于"在相邻句中的密度。

### 2.3 对话标签问题

| 类型 | 症状 | 建议 |
|------|------|------|
| 标签匮乏 | 只有"说/道" | 适当变化：低语、嘟囔、嘶吼、打断 |
| 标签过度 | "他咆哮道""她低语道""他咆哮道"循环 | 交替使用"说"和无标签段落 |
| 标签位置单一 | 全部在引语前 | 穿插在引语中/后/省略 |

### 2.4 段落长度均匀

**检测**：AI 倾向写出 3–5 行/段的均匀段落。手动浏览时，如果：
- 连续 10 个段落长度差不超过 2 行 → 标记
- 没有单行段落（用于节奏加速）
- 没有超过 10 行的段落（用于节奏放缓）

## 三、结构级检测

### 3.1 每章固定模式

AI 倾向的**模板化章节结构**：

```
起床/开场 → 日常小冲突 → 获得新信息 → 情绪波动 → 悬念结尾
```

**检测**：将 3–5 章的开头/高潮/结尾做成表格比较。如果结构雷同 > 70% → 重新编排章节节奏。

### 3.2 角色登场公式

AI 倾向的**角色描述模板**：

> X岁的Y职业，Z色的眼睛/头发，穿着一件……，嘴角挂着……的微笑，给人一种……的感觉。

**检测**：所有重要角色的首次出场是否都遵循"外貌→身份→印象"三段式？ → 打破它。有些角色可以只写行为，不写外貌。

### 3.3 章末悬念强迫症

**症状**：每一章最后一个句子都是一个悬念钩子。

> "就在这时，他看到了一个不可能出现的身影……"

**检测**：连续 5 章以上章末均为悬念句 → 偶尔在章节结尾留白或收束，给读者喘息空间。

## 四、示例对照

### 示例 1：AI 写法 → 改写

**AI 写法（有 AI 味）**：

> 在当下这个充满变数的时代，李明以一种近乎疲惫的方式推开了家门。空气中仿佛弥漫着一种说不清道不明的压抑气氛。他不禁让人感到，似乎某种改变即将发生。

**改写后**：

> 李明推开门。玄关的灯没开，客厅暗沉沉的。他把钥匙扔进碗里，叮当一声，没人回应。

### 示例 2：Tell → Show

**AI 写法**：

> 她感到很孤独。她觉得这个世界仿佛把她遗忘了。她似乎听到远处传来某种声音，但又不敢确定。

**改写后**：

> 手机屏幕亮了又暗，没有新消息。窗外的车声来了又走，每一辆都不是来接她的。她盯着天花板，数到第三百五十条裂缝时，楼下终于传来脚步声——却停在了隔壁门口。

### 示例 3：模板化开场 → 自然开场

**AI 写法**：

> 二十五岁的林晓是一名普通的图书管理员，她有一头柔顺的黑发和一双明亮的眼睛。她穿着一件简单的白色连衣裙，给人一种安静而温和的感觉。

**改写后**：

> 林晓在整理第三书架的旧书时，发现了一本不该出现在这里的书。

### 示例 4：章末悬念 → 变化节奏

**AI 写法**（连续三章结尾）：

> 第一章结尾：就在这时，门猛地被踹开了……
> 第二章结尾：他说出那句话的瞬间，所有人都沉默了……
> 第三章结尾：她看到了那张熟悉又陌生的脸……

**改写后**：

> 第一章结尾：门被踹开了。进来的人，她认识。
>
> 第二章结尾：房间里安静了很久。最后他开口说："你知道的，我没有选择。"
>
> 第三章结尾：月光照进来，在地板上铺成一条银色的路。她靠在窗边，什么也没等来。

## 五、检测流程建议

1. **扫描层**：用 ripgrep 搜索词汇级标记，高亮所有命中
2. **浏览层**：按章节随机抽读 3 页，检查句式单一、段落均匀、Show-Don't-Tell
3. **结构层**：做一章一行的摘要表，检查固定模式、登场公式、章末悬念
4. **修复层**：逐段改写，每次改完再扫一遍确认新 AI 味未引入

> **文无定法，检测清单是指标不是戒律。** 偶尔使用这些被标记的写法并无不可——问题在于**通篇如此**。
