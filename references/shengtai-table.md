# 声台形表 · 角色驱动写作方法论

> 适用于 autonovel-workflow Phase 1 Step 1.3（角色设计）和 Phase 2（初稿撰写）阶段
> 源自实战：2026-07 末日降临末世游戏 ch_01 人物卡重写

---

## 什么是声台形表

声台形表是演员训练的四大基本功，移植到小说创作中，用于确保角色在文本中的**立体感知**：

| 维度 | 小说中的含义 | 读者感知路径 |
|:----|:-------------|:------------|
| **声** | 角色的声音特质——音色、语速、语气习惯、停顿模式 | 听觉——读出声时的内心配音 |
| **台** | 角色的台词风格——什么时候说什么话、怎么说话 | 理性——读者判断角色智商和立场的依据 |
| **形** | 角色的肢体语言——站姿、手势、习惯性动作 | 视觉——读者在脑中构建的画面 |
| **表** | 角色的微表情——真假笑、生气信号、紧张泄漏 | 情感——读者判断角色真实情绪的窗口 |

**核心原则：** 四个维度不是装饰，是工具箱。写作时遇到「这个角色这里该怎么写」的问题，问自己：
- *他现在是什么声？*
- *他会说出什么样的台词/joke？*
- *他的身体在做啥？*
- *他脸上什么表情？*

答案不是靠猜，是靠人物卡里已经写好的声台形表清单。

## 人物卡中的声台形表（标准格式）

### 声 (Voice)
```
- 音色：普通青年音，不厚不薄。偏中低音区
- 语速：平时偏慢含混，认真时骤然清晰变快
- 语气习惯：紧张时用短句顿句，放松时用反问句
- 标志性停顿：说话前有半秒延迟——脑内预演
- 音量变化：不轻易抬高音量，需要时能吼——吼完立刻压回
```

**关键区分技巧：** 把角色的台词去掉角色名打印出来，看能不能凭语感分出是谁说的。分不出就是声没写够。

### 台 (Dialogue / 台词风格)
以场景为维度分格：

| 场景 | 风格 | 示例 |
|:----|:-----|:-----|
| 面对压力 | 短句+冷幽默 | 「房贷还没还完呢。不能死。」|
| 谈判周旋 | 反问句掌握主动权 | 「所以——你们从什么时候开始看我的？」|
| 被人看穿 | 自嘲化解尴尬 | 「我他妈连拖鞋都跑掉了一只。」|
| 说服别人 | 先共情再摊数据 | 「我知道你怕。但我算过了——」|

**好台词的检测标准：** 读者在对话中不需要「他说」「她说」标记也能知道是谁在说话。

### 形 (Physicality / 肢体语言)
```
- 站姿：微微偏侧，不正面迎人——给自己留观察角度
- 手的习惯：紧张时指关节敲大腿外侧（决策节拍器）
- 眼神：听人说话时看眉心，算东西时看左下方
- 动作节奏：平时慢，决定后快——无缝切换
- 标志性动作：做完高风险决策后低头看一眼自己的手
```

**好形体的检测标准：** 闭上眼能「看见」角色站在那。

### 表 (Expression / 微表情)
```
- 基本表情：平静偏冷的扑克脸——先想「该露什么表情」再露
- 真笑vs假笑：假笑嘴角动眼睛不动；真笑0.8秒后迅速收回
- 惊讶反应：眼皮先跳一下，0.3秒恢复平静——太快了让人觉得「你早知道？」
- 生气信号：不红脸不吼——眼皮下垂、嘴角拉平、呼吸变浅
```

**好表情的检测标准：** 读者能通过角色的表情变化读出剧情走向，不需要旁白解释「他很生气」。

## 声台形表 小纲标注格式

在 剧情小纲（小纲_第N章_章节名.txt）中，每段附加声台形表标注：

```
[[ 声台形表锚点 ]]
【声：说明该段冯枫的语音特征】
【台：核心台词策略/示例】
【形：关键肢体动作】
【表：关键微表情】
```

### 完整示例

```
■ Part 2 · 招募上门

【声台形表重点：圆滑世故+想方设法——开始展示人际操作的版本】

冯枫的反应不是直接开门——他先透过猫眼看了五秒。
【形：偏侧站姿，藏在门框后，只露半张脸。习惯性留观察角度。】

他开了门——不是被说服了，是他算过：能让他手机没信号的人，锁门没用。
【台：开门第一句话不是「你们是谁」——是「怎么进来的？」先抢主动权。】

陈先生展示位面投影。三段视频放完。
冯枫沉默十秒→「……再放一遍。」
【声：沉默时呼吸变深变慢，大脑在高速处理信息。】
【表：看完第三遍后眯了一下眼睛——不是怀疑，是「接受了但在找破绽」。】
```

### 全章节的情绪曲线标注
小纲末尾附加一条声台形表变化曲线：

```
■ 声台形表全章变化曲线：
  开篇：杀伐果断——动作快于思考的直觉型
  → 招募期：圆滑世故——用对话收集信息、测试对方底线
  → 签约期：想方设法——谈判、找漏洞、争取条款修改
  → 穿越后：内核浮现——恐惧但可控、用冷幽默和自我对话消化压力
```

## 何时使用声台形表

| 阶段 | 使用方式 | 
|:----|:---------|
| **Phase 1 Step 1.3（角色设计）** | 写人物卡时直接填入声台形表四个维度。一次填好，全周期可用 |
| **Phase 2（初稿撰写）** | 写每章前快速扫读角色声台形表，校准语调。写完后检查声台形表是否一致 |
| **Phase 3a V9-V16（角色一致性审计）** | 对照声台形表检查角色是否声音漂移——特别是「台」和「声」是否一致 |
| **小纲撰写** | 在每个场景段标记当前使用的声台形表锚点，防止走偏 |

## 常见坑

### ⚠️ 声台形表写太细（过度设计）
人物卡的声台形表不要超过 500 字。太细了写手记不住，等于没写。核心锚点每个维度 3-5 条足够。

### ⚠️ 小纲中每句都标注
小纲的声台形表标注应该是「锚点密度」——每段1-2个关键标注，不是每句一个。标注太多等于没有标注。

### ⚠️ 人物卡更新后小纲不同步
人物卡改了声台形表（如角色从「杀伐果断」变成「圆滑世故」），对应的小纲标注必须同步更新。否则写手按小纲写出来的是旧版人格。
